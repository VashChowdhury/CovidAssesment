# CovidAssesment
This code was from a culminating assesment based on loops and lists based on a covid data sets from the NY Times. 

The code answers the following questions:

When was the first positive COVID case in Rockingham County and Harrisonburg?

What day was the greatest number of new daily cases recorded in Harrisonburg and Rockingham County?

What was the worst seven day period in the city and county for new COVID cases?
